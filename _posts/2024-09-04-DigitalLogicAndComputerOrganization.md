# Course Introduction

<mark>Course number: 22000160 Credit: 4</mark>   

The teaching content of this course is as follows:   

<table border="1" width="100%"> <tr> <th>Section</th> <th>Content</th> </tr> <tr> <td>1. Binary Representation and Encoding</td> <td> <ul> <li>Number systems and binary representation</li> <li>Signed number representation</li> <li>Two’s complement and arithmetic properties</li> </ul> </td> </tr> <tr> <td>2. Fundamentals of Digital Logic</td> <td> <ul> <li>Boolean algebra and logic operations</li> <li>Logic gates and truth tables</li> <li>CMOS logic concepts and implementation overview</li> </ul> </td> </tr> <tr> <td>3. Combinational Logic Circuits</td> <td> <ul> <li>Design and analysis of combinational circuits</li> <li>Karnaugh maps and logic minimization</li> <li>Typical combinational components</li> </ul> </td> </tr> <tr> <td>4. Sequential Logic Circuits</td> <td> <ul> <li>Basic concepts of sequential logic</li> <li>Latches, flip-flops, and registers</li> <li>Timing, clocking, and state behavior</li> </ul> </td> </tr> <tr> <td>5. Introduction to Computer Organization</td> <td> <ul> <li>Basic structure of a digital computer</li> <li>Relationship between hardware components</li> <li>Overview of datapath and control concepts</li> </ul> </td> </tr> <tr> <td>6. Arithmetic Operations and Datapath Components</td> <td> <ul> <li>Binary arithmetic operations</li> <li>Addition, subtraction, multiplication, and division</li> <li>Arithmetic logic unit (ALU) design principles</li> </ul> </td> </tr> <tr> <td>7. Instruction Set Architecture (ISA)</td> <td> <ul> <li>Instruction formats and addressing modes</li> <li>Instruction execution process</li> <li>Role of ISA in hardware–software interface</li> </ul> </td> </tr> <tr> <td>8. Central Processing Unit: Overview</td> <td> <ul> <li>CPU components and functional units</li> <li>Datapath and control unit interaction</li> <li>Performance considerations</li> </ul> </td> </tr> <tr> <td>9. Single-Cycle CPU Design</td> <td> <ul> <li>Single-cycle datapath structure</li> <li>Control signal generation</li> <li>Instruction execution in a single cycle</li> </ul> </td> </tr> <tr> <td>10. Multi-Cycle CPU Design</td> <td> <ul> <li>Motivation for multi-cycle implementation</li> <li>Datapath reuse and control sequencing</li> <li>Performance comparison with single-cycle CPU</li> </ul> </td> </tr> <tr> <td>11. Pipeline CPU Fundamentals</td> <td> <ul> <li>Basic concepts of instruction pipelining</li> <li>Pipeline stages and datapath partitioning</li> <li>Throughput and speedup analysis</li> </ul> </td> </tr> <tr> <td>12. Pipeline Implementation</td> <td> <ul> <li>Pipeline control and instruction flow</li> <li>Instruction fetch and decode mechanisms</li> <li>Pipeline register design</li> </ul> </td> </tr> <tr> <td>13. Pipeline Hazards</td> <td> <ul> <li>Structural, data, and control hazards</li> <li>Hazard detection and resolution techniques</li> <li>Stalling and forwarding mechanisms</li> </ul> </td> </tr> <tr> <td>14. Advanced Pipeline Concepts</td> <td> <ul> <li>Pipeline optimization strategies</li> <li>Impact of hazards on performance</li> <li>Introduction to advanced pipeline designs</li> </ul> </td> </tr> <tr> <td>15. Course Review</td> <td> <ul> <li>Integration of digital logic and CPU design concepts</li> <li>Overall understanding of computer organization</li> <li>Preparation for final assessment</li> </ul> </td> </tr> </table>
